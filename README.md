# ✅ • Getting Started

> Follow these steps to run MinePlus on your system:

### 🚀 Installation

1. Clone the repository:

   ```
   git clone https://github.com/minimalharry/MinePlus.git
   ```

2. Go to project folder:

   ```
   cd MinePlus
   ```

3. Install dependencies:

   ```
   npm install
   ```

4. Configure bot token in `config.json`:

   ```json
   {
     "lang": "en",
     "token": "YOUR_BOT_TOKEN_HERE"
   }
   ```

5. Start the bot:

   ```
   node Index.js
   ```

# 🧩 • Systems Overview

> MinePlus is built with modular architecture:

* 🎫 Ticket System (setup, close, logs, rename, etc.)
* 🤖 AutoResponder (custom replies system)
* 👋 Welcome & Greeting events
* 🎨 Embed System (store, build, delete, list)
* 📜 Rules Panel System
* 🛠️ Admin & Moderation tools
* 🌍 Language system (EN, ES, PT supported)
